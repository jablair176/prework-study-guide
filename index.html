Git

  • `git status`: checks which branch we are currently working on
  • `git checkout -b branch name`: creates a new branch titled "branch name" and switches to it
  • `git add -A`: sends modifications to the staging area to be added in the next commit
  • `git commit -m "add discription of whats changing"`: creates the commit
  • `git pull origin main`: double-checking local branch is in sync with base branch
  • `git push origin branch name`: push changes we have locally to remote GitHub branch
